# 🚀 SmartRush: Hyper-local Quick Commerce & Logistics

SmartRush is a high-performance, full-stack MERN application designed to solve the challenges of hyper-local commerce. It integrates real-time store discovery via the Overpass API, a robust logistics engine for delivery partner assignment, and a premium glassmorphic UI.

---

## ✨ Key Features

- **📍 Real-time Store Discovery:** Leverages the Overpass API (OpenStreetMap) to dynamically fetch and map real-world retail outlets based on the user's live geolocation.
- **🚚 Intelligent Logistics Engine:** Automated delivery partner assignment and simulated live order tracking.
- **💎 Premium UI/UX:** A state-of-the-art glassmorphic design featuring a resizable dynamic sidebar, smooth Framer Motion animations, and full mobile responsiveness.
- **💳 Secure Payments:** Integrated Razorpay payment gateway with server-side HMAC signature verification.
- **📊 Business Intelligence:** Advanced analytics dashboard for store owners to track demand insights and performance metrics.
- **🌗 Dynamic Themes:** Seamless switching between high-contrast Dark and Light modes.

---

## 🛠️ Tech Stack

- **Frontend:** React.js, Vite, Framer Motion, Leaflet.js, Lucide Icons, Axios.
- **Backend:** Node.js, Express.js, JWT, Razorpay SDK.
- **Database:** MongoDB (Atlas), Mongoose ODM (Aggregation Pipelines).
- **Deployment:** Railway (Server), Vercel (Client).

---

## 🚀 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/chan570/smartrush.git
cd smartrush
```

### 2. Backend Setup
```bash
cd server
npm install
# Add your .env variables (MONGODB_URI, RAZORPAY_KEY, etc.)
npm start
```

### 3. Frontend Setup
```bash
cd client
npm install
npm run dev
```
